# desafio-front-end-contato-seguro

Desafio Front-End para empresa Contato Seguro. No desafio proposto, tive que desenvolver uma interface com React onde o usuário pudesse fazer um CRUD simples de livros e autores.

A página foi desenvolvida de acordo com os requisitos/funcionalidades listadas. Como requisitado, não foi utilizado bibliotecas de CSS, e além das imagens e ícones baixados, a estilização foi toda feita por mim (optei por não utilizar Radix UI).

Os dados cadastrados das entidades Livros e Autores estão sendo salvas no LocalStorage, de acordo com o pedido pelo desafio.
